# 💪 Fit Partner - Login

Es un login con registro y recuperar la contraseña, especialmente para un gimnasio, lo puedes implementar para tu tienda fitness, muy eficiente y rapido.

---

## 🚀 Características principales
- 🔐 **Login seguro** con validaciones  
- 📝 **Registro de usuarios** con:
  - Edad  
  - Estatura  
  - Peso  
- 👁️ Botón para mostrar/ocultar contraseña  
- 🎨 Diseño UI moderno inspirado en aplicaciones fitness  
- 📦 Arquitectura MVC  
- 🗄️ Base de datos MySQL  
- 🔄 Mensajes dinámicos de error y éxito  

---

## 🗂️ Tecnologías utilizadas
| Tecnología | Uso |
|-----------|-----|
| **PHP 8+** | Controladores y lógica de autenticación |
| **MySQL** | Almacenamiento de usuarios |
| **HTML5 / CSS3** | Interfaz y diseño responsivo |
| **Bootstrap Icons** | Icono de visibilidad de contraseña |
| **JavaScript** | Validaciones y eventos |
